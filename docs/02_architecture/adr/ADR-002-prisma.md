# ADR-002

## Title

Prisma ORM használata

## Status

Accepted

## Context

A projekt relációs adatbázist használ.

## Decision

Prisma ORM került kiválasztásra.

## Consequences

Előnyök:

- típusbiztos lekérdezések
- migrációkezelés
- gyors fejlesztés

Hátrányok:

- ORM absztrakciós réteg többletköltsége

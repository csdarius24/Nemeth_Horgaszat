# ADR-003

## Title

MySQL adatbázis használata

## Status

Accepted

## Context

A projekt strukturált relációs adatokat tárol.

## Decision

MySQL adatbázis használata.

## Consequences

Előnyök:

- széles körben támogatott
- stabil
- Hostinger kompatibilis

Hátrányok:

- horizontális skálázás korlátozottabb

# UX Flows

## Bejelentkezés

Felhasználó
→ Login oldal
→ Email + jelszó
→ Session létrehozás
→ Dashboard

---

## Új tó létrehozása

Dashboard
→ Halászat kiválasztása
→ Tavak
→ Új tó
→ Mentés
→ Tó megjelenik listában

---

## Telepítés

Dashboard
→ Tó kiválasztása
→ Telepítés
→ Halfaj
→ Darabszám
→ Mentés
→ Állomány frissül
→ Naplóbejegyzés készül

---

## Kivétel

Dashboard
→ Tó kiválasztása
→ Kivétel
→ Halfaj
→ Darabszám
→ Mentés
→ Állomány csökken
→ Naplóbejegyzés készül

---

## Etetés

Dashboard
→ Tó kiválasztása
→ Etetés
→ Mennyiség
→ Mentés
→ Esemény naplózódik
